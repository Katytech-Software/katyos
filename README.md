# Katy OS

Katy OS is an Arch Linux–based distribution built for developers and gamers who demand performance, control, and simplicity at the same time.  It combines a powerful and minimal core with a extensive driver support, and a seamless user experience—so you can focus on coding, playing, and creating instead of fighting your system.

# Katy OS Manifesto

We believe power should not come at the cost of usability.

Katy OS was created on a simple idea:
you shouldn’t have to fight your system to control it.

We stand on the shoulders of Arch Linux—embracing its flexibility, minimalism, and transparency—while removing the unnecessary friction that keeps people away.

# Our Principles

1. Control without complexity
Powerful systems should be intuitive, not punishing.
Katy OS delivers deep customization without forcing users into unnecessary complexity.

2. Built for creation and performance
Whether you’re writing code or pushing frames, your system should keep up.
No compromises between development and gaming.

3. Friendly, not fragile
A system can be easy to use without being limited.
Katy OS is approachable, but never restrictive.

4. Customization is a right
Your workflow is yours.
Keybindings, environment, behavior—everything should adapt to you.

5. Open means truly open
Powered by open-source and protected by GNU General Public License v3.0,
Katy OS will always belong to its community.

6. No unnecessary barriers
No gatekeeping. No elitism. No “RTFM culture” as a requirement.
You can learn, explore, and grow—without frustration.

# Our Goal

To be the system where:

- developers feel at home
- gamers get maximum performance
- and users stay in control

All at the same time.
